```
└────────────────┴────────────────┴───────────────────────────┘
```

## 1. 대시보드 탭

메인 화면으로, 플레이어 상태, 보스 상태, 루틴 관리, 시스템 로그를 한눈에 볼 수 있습니다.

### 레이아웃
```
┌─────────────────┬─────────────────────────────────────────┐
│  플레이어 상태  │                                         │
│  ┌───────────┐  │              보스 상태                  │
│  │이름/레벨  │  │  ┌──────────────────────────────────┐  │
│  │EXP 바    │  │  │  [보스 이름]                      │  │
│  │골드      │  │  │  [보스 이미지]                    │  │
│  │장비 효과  │  │  │  [보스 설명]                      │  │
│  └───────────┘  │  │  [HP 바]                         │  │
├─────────────────┤  └──────────────────────────────────┘  │
│   루틴 관리     │                                         │
│ ┌─────┐ ┌────┐ │                                         │
│ │생성 │ │목록│ │                                         │
│ └─────┘ └────┘ │                                         │
├─────────────────┼─────────────────────────────────────────┤
│  오늘의 루틴    │              시스템 로그                 │
│  ┌───────────┐  │  ┌──────────────────────────────────┐  │
│  │☐ 루틴 1  │  │  │[09:30] 루틴 완료: 아침 운동      │  │
│  │✔ 루틴 2  │  │  │[09:31] +20 EXP, +50 G           │  │
│  │☐ 루틴 3  │  │  │[09:31] 보스에게 4 데미지!       │  │
│  └───────────┘  │  └──────────────────────────────────┘  │
└─────────────────┴─────────────────────────────────────────┘
```

### 1.1 플레이어 상태 패널
| 요소 | 설명 |
|------|------|
| 이번 달 집중 공략 | 취약 루틴 이름 표시 (붉은색) |
| 이름/레벨 | 플레이어 이름과 현재 레벨 |
| EXP 바 | 현재 경험치 / 최대 경험치 |
| 골드 | 보유 골드 표시 |
| 인벤토리 버튼 | 인벤토리 창 열기 |
| 상점 버튼 | 아이템 상점 창 열기 |

### 1.2 보스 상태 패널
| 요소 | 설명 |
|------|------|
| 보스 이름 | 이번 달 보스의 이름 |
| 보스 이미지 | 보스 캐릭터 이미지 (350x350px) |
| 보스 설명 | 보스 컨셉 설명 텍스트 |
| HP 바 | 현재 HP / 최대 HP (빨간색) |

### 1.3 루틴 관리 패널
| 버튼 | 동작 |
|------|------|
| 루틴 생성 | 루틴 생성 창 열기 (RoutineManagerGUI) |
| 루틴 목록/수정/삭제 | 루틴 목록 창 열기 (RoutineListView) |

### 1.4 오늘의 루틴 패널
- 현재 날짜와 요일 표시
- 오늘 해당하는 루틴 목록
- 완료 상태 표시: `[✔ 완료]` / `[☐ 미완료]`
- 미완료 루틴이 상단, 완료 루틴이 하단으로 정렬

### 1.5 시스템 로그 패널
- 모든 게임 이벤트 기록
- 시간 스탬프 포함: `[HH:MM:SS] 메시지`
- 자동 스크롤 (최신 메시지가 하단)

## 2. 연속 달성 탭

루틴 수행 날짜를 달력 형태로 표시합니다.

### 레이아웃
```
┌─────────────────────────────────────────────────────────────┐
│         ◀  2025년 12월  ▶                                  │
├────┬────┬────┬────┬────┬────┬────┤
│ 일 │ 월 │ 화 │ 수 │ 목 │ 금 │ 토 │
├────┼────┼────┼────┼────┼────┼────┤
│    │  1 │  2 │  3 │  4 │ ⬛5│  6 │  ← 오늘(주황 테두리)
├────┼────┼────┼────┼────┼────┼────┤
│  7 │ 🟩8│ 🟩9│ 10 │ 11 │ 12 │ 13 │  ← 녹색: 루틴 완료
├────┼────┼────┼────┼────┼────┼────┤
│ 14 │ 15 │ 16 │ 17 │ 18 │ 19 │ 20 │
└────┴────┴────┴────┴────┴────┴────┘
```

### 색상 의미
| 색상 | 의미 |
|------|------|
| 🟩 녹색 배경 | 해당 날짜에 루틴 완료 |
| ⬛ 주황 테두리 | 오늘 날짜 |
| 🔵 파란색 텍스트 | 토요일 |
| 🔴 빨간색 텍스트 | 일요일 |

### 탐색
- ◀ 버튼: 이전 월로 이동
- ▶ 버튼: 다음 월로 이동

## 3. 월간 통계 탭

이번 달 루틴 수행 통계를 표시합니다.

### 레이아웃
```
┌─────────────────────────────────────────────────────────────┐
│              2025년 12월 리포트                             │
│         이번 달 총 15일 루틴을 수행했습니다!                │
│  ████████████████░░░░░░░░░░░░░░░░  월간 달성률: 48%        │
├─────────────────────────────────────────────────────────────┤
│                    태그별 수행 현황                          │
│                                                             │
│      ████  ██████  ████████  ████                          │
│       15     25       35      20                            │
│      공부   운동     생활    취미                           │
├─────────────────────────────────────────────────────────────┤
│                  ▼ 오늘의 한마디 ▼                          │
│              좋은 흐름이에요! 계속 화이팅!                   │
├─────────────────────────────────────────────────────────────┤
│  ┌─────────────────────────────────────────────────────┐   │
│  │ 상세 분석                                           │   │
│  │ • 가장 많이 수행한 태그: 생활 (35회)               │   │
│  │ • 가장 적게 수행한 태그: 공부 (15회)               │   │
│  └─────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────┘
```

### 구성 요소
| 요소 | 설명 |
|------|------|
| 월 제목 | 현재 연/월 표시 |
| 달성 일수 | 루틴을 수행한 총 일수 |
| 진행률 바 | 달성 일수 / 월 총 일수 (백분율) |
| 막대그래프 | 태그별 수행 횟수 시각화 |
| 격려 메시지 | 달성률에 따른 동기부여 메시지 |
| 상세 분석 | 태그별 분석 텍스트 |

## 4. 루틴 생성 창

새 루틴을 생성하는 폼입니다.

### 레이아웃
```
┌─────────────────────────────────────────┐
│  MRA --- 루틴 생성/관리                 │
├─────────────────────────────────────────┤
│  루틴 이름: [________________]          │
│                                         │
│  태그: [공부 ▼]                        │
│                                         │
│  반복 요일: ☐월 ☐화 ☐수 ☐목 ☐금 ☐토 ☐일 │
│                                         │
│      [루틴 생성]  [루틴 목록 보기]       │
└─────────────────────────────────────────┘
```

### 입력 필드
| 필드 | 타입 | 필수 여부 |
|------|------|----------|
| 루틴 이름 | 텍스트 입력 | 필수 |
| 태그 | 드롭다운 선택 | 필수 (기본값 있음) |
| 반복 요일 | 체크박스 | 1개 이상 필수 |

## 5. 루틴 목록 창

요일별 루틴 목록을 표시합니다.

### 레이아웃
```
┌─────────────────────────────────────────────────────────────┐
│  MRA --- 루틴 목록                                          │
├────┬────┬────┬────┬────┬────┬────┤
│ 월 │ 화 │ 수 │ 목 │ 금 │ 토 │ 일 │  ← 요일 탭
├────┴────┴────┴────┴────┴────┴────┤
│ ┌────┬─────────────────┬──────┬────┐ │
│ │완료│ 루틴 이름       │ 태그 │ ID │ │  ← 테이블 헤더
│ ├────┼─────────────────┼──────┼────┤ │
│ │ ☐ │ 아침 운동       │ 운동 │(숨김)│ │
│ │ ☐ │ 독서 30분       │ 공부 │(숨김)│ │
│ │ ☑ │ 물 마시기       │ 생활 │(숨김)│ │  ← 회색 배경 (완료됨)
│ └────┴─────────────────┴──────┴────┘ │
└─────────────────────────────────────────────────────────────┘
```

### 기능
| 동작 | 결과 |
|------|------|
| 체크박스 클릭 | 루틴 완료 처리 |
| 우클릭 | 수정/삭제 메뉴 표시 |
| 탭 클릭 | 해당 요일 루틴 표시 |

### 완료 상태 표시
| 상태 | 배경색 | 텍스트 색 | 체크박스 |
|------|--------|----------|----------|
| 미완료 | 흰색 | 검정 | 활성화 |
| 완료됨 | 회색 | 검정 | 비활성화 |
| 과거/미래 요일 | 흰색 | 회색 | 비활성화 |

## 6. 인벤토리 창

소유 아이템과 장착 장비를 관리합니다.

### 레이아웃
```
┌─────────────────────────────────────────────────────────────┐
│  플레이어 인벤토리 - [플레이어 이름]                        │
├─────────────────────────────────────────────────────────────┤
│  [이름] (Lv.5) | 골드: 1250 G                              │
│  적용 효과: 공격력 +15 / 추가경험치 +5                      │
├────────────────┬────────────────────────────────────────────┤
│ 착용 중인 장비 │ 가방 (인벤토리)                           │
├────────────────┴────────────────────────────────────────────┤
│  ┌──────┬────────────┬──────────┐                          │
│  │ 부위 │ 아이템 이름 │ 효과     │                          │
│  ├──────┼────────────┼──────────┤                          │
│  │ 무기 │ 무거운 철검 │ 공격력+15│                          │
│  │ 머리 │ 가죽 모자   │ 경험치+5 │                          │
│  └──────┴────────────┴──────────┘                          │
│                    [선택 장비 해제]                         │
├─────────────────────────────────────────────────────────────┤
│                        [닫기]                               │
└─────────────────────────────────────────────────────────────┘
```

### 탭 구성
| 탭 | 내용 |
|----|------|
| 착용 중인 장비 | 현재 장착된 장비 목록, 해제 버튼 |
| 가방 | 보유 중인 모든 아이템, 장착 버튼 |

## 7. 아이템 상점 창

아이템을 구매하는 상점입니다.

### 레이아웃
```
┌─────────────────────────────────────────────────────────────┐
│  아이템 상점                                                │
├─────────────────────────────────────────────────────────────┤
│      어서오세요! 없는 것 빼고 다 있는 상점입니다.           │
│                내 보유 골드: 1250 G                         │
├─────────────────────────────────────────────────────────────┤
│  ┌────────────┬──────┬────────────┬────────┐               │
│  │ 아이템 이름 │ 부위 │ 효과       │ 가격   │               │
│  ├────────────┼──────┼────────────┼────────┤               │
│  │ 수련용 목검 │ 무기 │ 공격력 +5  │ 50 G   │               │
│  │ 무거운 철검 │ 무기 │ 공격력 +15 │ 200 G  │               │
│  │ 용사의 검   │ 무기 │ 공격력 +50 │ 1000 G │               │
│  │ 가죽 모자   │ 머리 │ 경험치 +5  │ 100 G  │               │
│  └────────────┴──────┴────────────┴────────┘               │
├─────────────────────────────────────────────────────────────┤
│              [선택한 아이템 구매하기]                       │
└─────────────────────────────────────────────────────────────┘
```

### 구매 흐름
1. 테이블에서 아이템 선택 (클릭)
2. "구매하기" 버튼 클릭
3. 확인 팝업에서 "예" 선택
4. 골드 차감 및 인벤토리에 추가
5. 골드 표시 갱신

## 8. 팝업 및 다이얼로그

### 8.1 캐릭터 생성 팝업
- 최초 실행 시 표시
- 플레이어 이름 입력
- 취소 시 기본값 "루틴 수행자" 사용

### 8.2 취약 루틴 설정 팝업
- 매월 1일 또는 취약 루틴 미설정 시 표시
- 이번 달 집중할 루틴 이름 입력

### 8.3 보스 스토리 팝업
- 해피 엔딩: 보스 처치 성공 시
- 배드 엔딩: 월 변경 시 미처치 보스 있을 때

### 8.4 과거/미래 루틴 경고 팝업
- 과거 또는 미래 요일의 루틴 체크 시도 시
- "X요일은 체크할 수 없습니다" 메시지
