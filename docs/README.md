# 🎮 My Routine Adventure (MRA)

**나만의 루틴 RPG** - 일상 루틴을 게임처럼 관리하는 Java Swing 데스크톱 애플리케이션

## 📌 프로젝트 개요

My Routine Adventure(MRA)는 사용자의 일상 루틴을 게임화(Gamification)하여 동기 부여를 제공하는 데스크톱 애플리케이션입니다. 루틴을 완료하면 경험치와 골드를 얻고, 매월 등장하는 보스를 처치하며, 아이템을 수집하고 장착하는 RPG 요소를 결합했습니다.

## ✨ 주요 기능

### 1. 루틴 관리
- 루틴 생성/수정/삭제 (CRUD)
- 요일별 반복 설정 (월~일)
- 태그 분류 (공부, 운동, 생활, 취미, 기타)
- 완료 상태 추적 및 자동 정렬

### 2. RPG 시스템
- **레벨 시스템**: 루틴 완료 시 경험치 획득, 자동 레벨업
- **골드 시스템**: 루틴 완료 시 골드 획득, 상점에서 아이템 구매
- **보스 시스템**: 매월 새로운 보스 등장, 루틴 완료로 데미지 누적
- **취약 루틴**: 이번 달 집중 공략 루틴 설정 시 보상 2배

### 3. 아이템 시스템
- 루틴 완료 시 아이템 드랍 (확률 기반)
- 장비 슬롯: 머리, 몸통, 다리, 무기, 기타
- 장비 효과: 공격력 보너스, 경험치 보너스
- 아이템 상점에서 장비 구매

### 4. 통계 및 시각화
- 월간 달성률 표시
- 태그별 수행 현황 막대그래프
- 연속 달성 현황 달력 (스트릭)

## 🛠 기술 스택

| 구분 | 기술 |
|------|------|
| 언어 | Java SE 8+ |
| GUI | Java Swing |
| 데이터 저장 | 텍스트 파일 (루틴), Java 직렬화 (플레이어, 보스) |
| 빌드 | javac (수동 컴파일) |

## 📁 프로젝트 구조

```
My-Routine-Adventure/
├── J1103/
│   ├── src/J1103/          # 소스 코드
│   │   ├── MainDashboard.java      # 메인 진입점 및 대시보드 UI
│   │   ├── RoutineManager.java     # 루틴 비즈니스 로직 (핵심)
│   │   ├── Routine.java            # 루틴 데이터 모델
│   │   ├── Player.java             # 플레이어 데이터 모델
│   │   ├── Boss.java               # 보스 데이터 모델
│   │   ├── Item.java               # 아이템 데이터 모델
│   │   ├── Inventory.java          # 인벤토리 관리
│   │   ├── FileManager.java        # 파일 입출력
│   │   ├── DateUtil.java           # 시스템 시간 유틸리티
│   │   └── ...                     # 기타 UI 클래스들
│   ├── bin/                # 컴파일된 클래스 파일
│   └── docs/               # 프로젝트 내부 문서
├── docs/                   # 프로젝트 루트 문서 (현재 위치)
├── routines_data.txt       # 루틴 데이터 저장 파일
├── player_data.dat         # 플레이어 데이터 (직렬화)
└── boss_data.txt           # 보스 데이터 (직렬화)
```

## 🚀 실행 방법

### 컴파일
```bash
cd J1103
javac -encoding UTF-8 -d bin src/J1103/*.java
```

### 실행
```bash
java -cp bin J1103.MainDashboard
```

## 📊 데이터 파일 형식

### routines_data.txt
```
ID|이름|태그|요일목록|완료날짜Map
R1|아침 운동|운동|월,화,수,목,금|월:2025-12-05;화:2025-12-03
```

### player_data.dat / boss_data.txt
- Java 직렬화 형식 (ObjectOutputStream)
- 프로그램 종료 시 자동 저장
