## 📱 1. 메인 대시보드 (MainDashboard)

프로그램의 메인 화면으로, 3개의 탭으로 구성되어 있습니다.

### 탭 구조
```
┌────────────────────────────────────────────────────────────────┐
│  [✅ 대시보드]  [🔥 연속 달성 현황]  [📊 월간 통계]            │
├────────────────────────────────────────────────────────────────┤
│                                                                │
│                       탭 컨텐츠 영역                            │
│                                                                │
└────────────────────────────────────────────────────────────────┘
```

### 컴포넌트 설명

| 영역 | 클래스/메서드 | 설명 |
|------|--------------|------|
| 플레이어 상태 | `createPlayerStatusPanel()` | 이름, 레벨, 골드, 경험치바 표시 |
| 루틴 관리 | `createRoutineManagementPanel()` | 루틴 생성/목록 버튼 |
| 보스 상태 | `createBossStatusPanel()` | 월별 보스 이미지/HP 표시 |
| 오늘의 루틴 | `createDateRoutinePanel()` | 오늘 날짜의 루틴 목록 |
| 시스템 로그 | `createLogPanel()` | 실시간 시스템 로그 |

## 📋 2. 루틴 목록 화면 (RoutineListView)

요일별 탭으로 루틴을 관리합니다.

```
┌────────────────────────────────────────────────────────────────┐
│          MRA --- 루틴 목록                              [X]    │
├────────────────────────────────────────────────────────────────┤
│  [월]  [화]  [수]  [목]  [금]  [토]  [일]                       │
├────────────────────────────────────────────────────────────────┤
│ ┌────────┬──────────────────────────────┬────────────┐        │
│ │  완료   │         루틴 이름            │    태그    │        │
│ ├────────┼──────────────────────────────┼────────────┤        │
│ │   [✓]   │   독서                       │   공부     │        │
│ ├────────┼──────────────────────────────┼────────────┤        │
│ │   [ ]   │   운동                       │   운동     │        │
│ ├────────┼──────────────────────────────┼────────────┤        │
│ │   [ ]   │   명상                       │   생활     │        │
│ └────────┴──────────────────────────────┴────────────┘        │
│                                                                │
│  * 완료된 루틴은 회색으로 표시되며 체크박스가 비활성화됩니다.     │
│  * 우클릭 시 수정/삭제 메뉴가 표시됩니다.                       │
└────────────────────────────────────────────────────────────────┘
```

### 상호작용
| 동작 | 결과 |
|------|------|
| 체크박스 클릭 | 루틴 완료 처리 (경험치/골드 획득) |
| 우클릭 | 수정/삭제 팝업 메뉴 표시 |
| 탭 클릭 | 해당 요일 루틴 목록 표시 |

## 🆕 3. 루틴 생성 화면 (RoutineManagerGUI)

새 루틴을 생성하는 입력 폼입니다.

```
┌────────────────────────────────────────────────────────────────┐
│          MRA --- 루틴 생성/관리                         [X]    │
├────────────────────────────────────────────────────────────────┤
│                                                                │
│  루틴 이름:  [____________________________________]            │
│                                                                │
│  태그:       [공부        ▼]                                   │
│              ├─ 공부                                           │
│              ├─ 운동                                           │
│              ├─ 생활                                           │
│              ├─ 취미                                           │
│              └─ 기타                                           │
│                                                                │
│  반복 요일:  [ ]월  [ ]화  [ ]수  [ ]목  [ ]금  [ ]토  [ ]일   │
│                                                                │
│            [루틴 생성]    [루틴 목록 보기]                      │
│                                                                │
└────────────────────────────────────────────────────────────────┘
```

### 입력 필드 유효성 검사
- 루틴 이름: 필수 (빈 문자열 불가)
- 반복 요일: 최소 1개 이상 선택 필수

## 🏪 6. 상점 화면 (ShopView)

아이템을 구매합니다.

```
┌────────────────────────────────────────────────────────────────┐
│                    아이템 상점                          [X]    │
├────────────────────────────────────────────────────────────────┤
│          어서오세요! 없는 것 빼고 다 있는 상점입니다.           │
│                                                                │
│                  내 보유 골드: 350 G                           │
├────────────────────────────────────────────────────────────────┤
│ ┌─────────────────┬────────┬─────────────────┬──────────┐     │
│ │   아이템 이름    │  부위  │       효과      │   가격   │     │
│ ├─────────────────┼────────┼─────────────────┼──────────┤     │
│ │  수련용 목검     │  무기  │   공격력 +5     │   50 G   │     │

│ │  가죽 모자       │  머리  │   경험치 +5     │  100 G   │     │
│ │  학자의 안경     │  머리  │   경험치 +10    │  300 G   │     │
│ │  여행자의 옷     │  몸통  │   경험치 +5     │  150 G   │     │
│ │  기사의 갑옷     │  몸통  │   공격력 +20    │  500 G   │     │
```

1. 테이블에서 아이템 선택

3. 확인 다이얼로그 표시
4. 골드 충분 시: 구매 성공, 인벤토리에 추가
5. 골드 부족 시: 구매 실패 메시지

## 🎨 UI 컴포넌트 사용

### 주요 Swing 컴포넌트
| 컴포넌트 | 용도 |
|----------|------|
| `JFrame` | 메인 창, 서브 창 |
| `JDialog` | 수정 다이얼로그 (모달) |
| `JTabbedPane` | 탭 패널 |
| `JTable` | 루틴/아이템 목록 |
| `JProgressBar` | 경험치바, 보스 HP바 |
| `JList` | 오늘의 루틴 목록 |
| `JTextArea` | 시스템 로그, 상세 분석 |
| `JComboBox` | 태그 선택 |
| `JCheckBox` | 반복 요일 선택 |

### 레이아웃 관리자
| 레이아웃 | 사용처 |
|----------|--------|
| `BorderLayout` | 메인 프레임, 패널 기본 구조 |
| `GridBagLayout` | 대시보드 그리드 배치 |
| `GridLayout` | 달력, 버튼 패널 |
| `FlowLayout` | 버튼 그룹, 체크박스 그룹 |

## 🔔 사용자 피드백

### 다이얼로그 메시지
| 상황 | 메시지 타입 | 예시 |
|------|------------|------|
| 성공 | `INFORMATION_MESSAGE` | "루틴이 생성되었습니다." |
| 경고 | `WARNING_MESSAGE` | "이름과 반복 요일을 모두 선택해주세요." |
| 오류 | `ERROR_MESSAGE` | "골드가 부족합니다!" |
| 확인 | `showConfirmDialog` | "정말 삭제하시겠습니까?" |

### 시스템 로그 포맷
```
[HH:MM:SS] 메시지 내용
```

예시:
```
[09:30:00] 프로그램 시작. (이전 데이터 로드)
[09:35:22] ✅ 루틴 완료: 독서
[09:35:22] 🎉 **아이템 획득!** 행운의 동전이(가) 인벤토리에 추가되었습니다.
[09:40:15] ➕ 루틴 생성: 운동 (ID: R5)
```
